### Installation
```bash
$ [sudo] pip install npm-register-cli
```

```bash
$ [sudo] npm i -g npm-register-cli
```

#### Features
+   register npmjs.com package

#### Examples
```bash
$ npm-register name1 name2
https://www.npmjs.com/package/name1
https://www.npmjs.com/package/name2
```
